## ch1 Relational Databases

### Databases

這堂課程的目標: 

1. 電子化儲存 data
2. 使用 sql code 儲存資料，撈取資料

- database vs traditional data stored 

database 的優勢為:

1. 更好的儲存空間比起舊的sheet 存放的方式
2. 儲存更安全，因為加密的關係
3. 可以同時多個 user 使用資料庫進行撈取讀取資料庫訊息
